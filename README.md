# auto-summary-whale

유튜브를 시청할 때 작은 버튼이나 단축키로 작동하는 웨일 확장입니다. 버튼을 누르면 웨일의 듀얼탭 기능으로 Gemini 웹사이트를 나란히 열고, 현재 보고 있는 유튜브 URL을 자동으로 입력합니다. API를 사용하지 않습니다.

## 주요 기능

- 개별 영상 제목 오른쪽에 "Gemini" 버튼 표시
- **버튼 클릭** 시: 웨일 듀얼탭(화면 분할)에 Gemini 열기
- **단축키** `Ctrl+Shift+U` (mac: `Command+Shift+U`): 일반 새 탭에 Gemini 열기 (듀얼탭 아님)
- URL 자동 입력 옵션 제공

## 듀얼탭과 단축키

듀얼탭(좌우 화면 분할)은 **버튼 클릭**으로만 사용할 수 있습니다. 단축키는 사용자 제스처 제약으로 인해 일반 새 탭에 Gemini를 열며, URL 자동 입력은 그대로 적용됩니다.

## 설치 방법

1. 이 프로젝트를 클론하거나 zip으로 내려받아 압축을 풉니다.
2. 웨일에서 `whale://extensions`로 이동합니다.
3. 우측 상단의 "개발자 모드"를 켭니다.
4. "압축해제된 확장 프로그램을 로드합니다"를 선택하고 이 프로젝트 폴더를 지정합니다.
